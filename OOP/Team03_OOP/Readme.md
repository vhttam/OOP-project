Ứng Dụng Quản Lý Siêu Thị Mini

- Mô Tả Dự Án : Ứng dụng này giúp quản lý hàng hóa, bán hàng và theo dõi kho cho siêu thị mini. Nó hỗ trợ thêm/sửa/xóa sản phẩm, tạo hóa đơn bán hàng, cảnh báo hết hàng và tạo báo cáo doanh thu. Backend phát triển bằng Java (Spring Boot), frontend bằng HTML, CSS và JavaScript thuần, phù hợp cho siêu thị nhỏ lẻ.

- Tính Năng Chính
+ Quản Lý Sản Phẩm: Thêm, sửa, xóa sản phẩm (tên, giá, số lượng, danh mục).
+ Quản Lý Bán Hàng: Tạo hóa đơn, tính tổng tiền, áp dụng giảm giá, in hóa đơn.
+ Theo Dõi Kho: Cảnh báo khi sản phẩm sắp hết, nhập/xuất kho tự động.
+ Báo Cáo: Thống kê doanh thu theo ngày/tháng, sản phẩm bán chạy.
+ Phân Quyền: Tài khoản cho nhân viên (bán hàng) và quản lý (toàn quyền).


- Công Nghệ Sử Dụng
+ Backend: Java với Spring Boot (cho API REST, xử lý logic và database).
+ Frontend: HTML, CSS, JavaScript thuần (hoặc với jQuery nếu cần).
+ Database: MySQL hoặc PostgreSQL (tích hợp với Spring Boot).
+ Công Cụ Khác: Maven (cho build Java), Git, Docker (tùy chọn).
+ Cài Đặt và Chạy Ứng Dụng

- Yêu Cầu Hệ Thống
+ Java JDK (phiên bản 11+)
+ Maven (hoặc Gradle)
+ Database: MySQL/PostgreSQL (cài đặt và chạy riêng)
+ Git
